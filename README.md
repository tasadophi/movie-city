# Movie City Api

This is a simple backend project that developed by node and express. Also it uses mongo db for database. <br/> The api postman docs file is in **docs** directory.

# Usage:

Its really simple. Just follow steps bellow:

1- Clone the repo.

2- Config MongoDB on your machine/server or create your db on cloud.

3- Create `.env` file in the root of project and set this variables:

    PORT=8000
    DB_URL=<YOUR_DB_ADDRESS>

4- Install NodeJs (at least 20 LTS version) + npm.

5- Run `npm i`.

6- Run `npm run build`.

7- Run `npm start`, server will listen on port 8000. Also sampleData will add to DB.
